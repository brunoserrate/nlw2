Proffy é uma plataforma para conectar alunos e professores de matérias diversas.

## Instalação

Use o comando:

```sh
npm install
```
Para instalar as dependências de cada pasta

## Inicializando

Para inicializar cada modulo:

```sh
npm start
```
Portas:
* Server: 3333 (server.ts);
* Mobile: expo default;
* Web: 3000 (react default);
